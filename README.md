# BelgeArsivYonetimSistemi

## Proje Temel Amacı
Fiziksel ortamda bulunan evrakların emanet işlemleri ve konum bilgilerini tutacak şekilde dijital ortama taşınması ve
emanet geçmişi gibi verilerin tutulmasıdır.
